## Overview

Agentic Radar is a security scanning tool designed specifically for agentic AI workflows to identify and report potential vulnerabilities.

## Features

- Scans agentic workflow source code for security issues
- Finds vulnerabilities including CVE and OWASP categories
- Generates interactive reports for visualization
- Suggests remediation steps for identified issues
- Supports popular agentic workflow frameworks
