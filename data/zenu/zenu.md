## Overview

zenu is a deep learning framework written entirely in Rust, providing native training capabilities with automatic differentiation and support for CPU and CUDA hardware acceleration.

## Features

- Native Rust deep learning framework
- Training support with gradient-based optimization
- Automatic differentiation (AutoDiff) for backpropagation
- CPU and CUDA backend support

## Backend Support

- CPU: Computation via cblas and openblas-src
- CUDA: NVIDIA GPU acceleration via manual CUDA bindings

## Dependencies

- cblas: C BLAS interface
- openblas-src: OpenBLAS linear algebra library
- Manual bindings to CUDA
