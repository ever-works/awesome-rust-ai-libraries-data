## Overview

LiveRecall is an open-source tool for capturing, indexing, and retrieving screen activity snapshots using semantic search.

## Features

- Screen snapshot capture and indexing
- Semantic search for activity retrieval
- Encryption for privacy
- AI agent assistance for research
- Task augmentation based on personal activity history
- Open-source alternative to Microsoft Recall

## Use Cases

- Personal productivity enhancement
- Research and content creation assistance
- Journaling and blog post creation
- Activity-based knowledge retrieval
