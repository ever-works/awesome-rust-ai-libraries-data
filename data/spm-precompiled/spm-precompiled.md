## Overview

SPM Precompiled is a specialized Rust crate that parses SentencePiece models (SPM format) without requiring the C++ SentencePiece library at runtime. This enables pure-Rust deployment of SentencePiece-based tokenization.

## Features

- Parses SentencePiece SPM model files in pure Rust
- No C++ library dependency at runtime
- Lightweight and easy to embed in Rust applications
- Compatible with pretrained SentencePiece models
