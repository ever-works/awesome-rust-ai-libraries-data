## Overview

The `fann` crate provides Rust bindings for the Fast Artificial Neural Network (FANN) library, a free open source neural network library originally written in C. It wraps the FANN C API to provide a safe and idiomatic Rust interface.

## Features

- Wrapper around the established FANN C library
- Feedforward neural network training and inference
- Multiple backpropagation algorithms
- Activation function support
- Model serialization and loading
