## Overview

Open Interpreter is a coding agent enabling language models to execute code locally, facilitating natural-language interaction with your computer's capabilities while overcoming limitations of hosted solutions.

## Features

- Local code execution with language models
- Interactive and programmatic chat modes
- System message customization
- Computer keyboard and mouse control
- No internet access or package restrictions of hosted solutions
- Enhanced control and flexibility for development

## Pricing

Free and open-source.
