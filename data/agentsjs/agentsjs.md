## Overview

Agents.js is a JavaScript framework designed for building AI agents that run natively in the browser with event-driven architecture.

## Features

- Browser-native implementation for client-side agent execution
- Event-driven architecture for responsive agent behavior
- Tool abstraction layer for easy capability extension
- Memory management for stateful agent interactions
- Real-time processing capabilities for live interactions
