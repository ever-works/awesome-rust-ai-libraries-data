## Overview

DramaLlama provides an ergonomic, high-level Rust API built on top of llama.cpp, specifically focused on chat completion use cases. It abstracts away the complexity of managing conversation context and model inference into a simple, intuitive interface.

## Features

- High-level API specifically designed for chat completions
- Built on llama.cpp for efficient local model inference
- Ergonomic Rust interface simplifying conversational AI development
- Manages conversation context and message formatting
