## Overview

Basaran provides an OpenAI-compatible API layer for Hugging Face Transformer models, allowing developers to use open-source models with the same interface as OpenAI's text completion API.

## Features

- OpenAI-compatible streaming API
- Works with any Hugging Face Transformers model
- Real-time text streaming
- No dependency on proprietary APIs
- Self-hosted deployment

## Pricing

Free and open-source.
