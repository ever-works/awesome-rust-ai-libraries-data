## Overview

Allms is a unified client library that abstracts over multiple LLM APIs. It provides a single, common interface for interacting with OpenAI, Anthropic, Mistral, and other providers, enabling provider-agnostic application development.

## Features

- Single interface for multiple LLM API providers
- Supports OpenAI, Anthropic, Mistral, and more
- Common chat and completion traits
- Easy provider switching without code changes
