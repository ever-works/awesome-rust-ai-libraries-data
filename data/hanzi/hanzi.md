## Overview

Hanzi is an open-source MCP server and Chrome extension that enables AI agents to interact with a real signed-in browser for comprehensive task automation.

## Features

- MCP server integration
- Chrome extension for browser access
- Task-level automation via single tool calls
- Built-in skills:
  - LinkedIn prospecting
  - End-to-end testing
  - Social media posting
- Compatible with Claude Code, Cursor, Codex, and Windsurf
- Quick setup: npx hanzi-in-chrome setup

## Architecture

- MCP server for agent communication
- Chrome extension for browser control
- Pre-built skill library
