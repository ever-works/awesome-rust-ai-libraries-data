## Overview

KubeAI is a Kubernetes operator that simplifies serving ML models in production. It provides a unified interface for serving various model types including VLMs, LLMs, embeddings, and speech-to-text models.

## Features

- Kubernetes-native operator architecture
- Supports VLMs, LLMs, and embeddings
- Speech-to-text model support
- Auto-scaling and resource management
- Simplified deployment workflows
