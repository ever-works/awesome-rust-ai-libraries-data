## Overview

Upsonic is a reliability-focused agent framework that supports the Model Context Protocol (MCP) and provides isolated environments for safe agent execution.

## Features

- Easy-to-activate reliability layers for production stability
- Model Context Protocol (MCP) support
- Integrated Browser Use and Computer Use capabilities
- Isolated environment to run agents safely
- Task-Centric Design for focused agent workflows
