## Overview

An unofficial Rust SDK for the OpenAI API that enables Rust applications to interact with OpenAI's services including GPT models and embeddings. Built with asynchronous support and mirrors the official API endpoints.

## Features

- Supports completions and chat completions endpoints
- Asynchronous API using Tokio
- Mirrors official OpenAI API endpoints
- Supports embeddings, image generation, and more
