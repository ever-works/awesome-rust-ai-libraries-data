## Overview

mmnn is a Rust-based CLI tool that provides neural network propagation and backpropagation capabilities. Networks are configured through JSON configuration files, offering a straightforward command-line approach to neural network experimentation.

## Features

- Neural network forward propagation
- Backpropagation for gradient computation
- JSON-based configuration for defining network architectures
- CLI interface for easy invocation
- Lightweight and dependency-free

## Configuration

- Networks defined via JSON configuration files
- Command-line argument support for runtime parameters

## Backend Support

- No specific hardware backend; runs on CPU

## Pricing

Free and open-source.
