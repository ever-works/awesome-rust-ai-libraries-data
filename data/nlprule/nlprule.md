## Overview

NLPRule is a rule-based text preprocessing library written in Rust, focusing on grammar and spelling correction. It offers fast, low-resource text processing without the need for ML models, making it suitable for lightweight NLP pipelines.

## Features

- Rule-based grammar and spelling correction
- Fast and low-resource usage profile
- No ML model dependencies required
- Suitable for text preprocessing in NLP pipelines
- Designed for production use with minimal overhead
