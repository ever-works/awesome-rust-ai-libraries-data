## Overview

Modus is an open-source, serverless framework for building intelligent agents and APIs.

## Features

- Serverless deployment architecture
- Support for Go programming language
- Support for AssemblyScript (TypeScript-like)
- WebAssembly-based execution
- Intelligent agent scaffolding
- API creation tools
- Lightweight runtime
- Decentralized computation model

## Pricing

Free and open-source.
