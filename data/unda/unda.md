## Overview

unda is a general-purpose machine learning crate for Rust that compiles to XLA (Accelerated Linear Algebra), providing a pathway for efficient ML computation across diverse hardware platforms.

## Features

- Compiles ML computations to XLA intermediate representation
- Supports XLA backend for hardware-accelerated execution
- General-purpose ML operations and primitives
- Leverages xla-rs bindings for XLA integration

## Backend Support

- XLA (Accelerated Linear Algebra)

## Dependencies

- xla-rs: Rust bindings for XLA
