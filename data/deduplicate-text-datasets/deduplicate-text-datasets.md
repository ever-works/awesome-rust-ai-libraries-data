# deduplicate-text-datasets

A repository containing code to deduplicate language model datasets as described in the paper 'Deduplicating Training Data Makes Language Models Better'. Includes the ExactSubstr deduplication implementation written in Rust, used to improve LM training data quality by removing duplicate examples.

[https://github.com/google-research/deduplicate-text-datasets](https://github.com/google-research/deduplicate-text-datasets)
