## Overview

Xinference is a unified inference platform that allows users to run inference with any open-source language models, speech recognition models, and multimodal models. It supports deployment in the cloud, on-premises, or on a laptop.

## Features

- Single line of code to replace OpenAI APIs with other LLMs
- Supports open-source language models
- Speech recognition model support
- Multimodal model support
- Cloud, on-premises, and local deployment
- RESTful API with OpenAI-compatible interface
