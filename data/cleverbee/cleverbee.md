## Overview

CleverBee is a deep research assistant agent designed for automated web research with interactive UI, token tracking, and multi-LLM support.

## Features

- Interactive web UI via Chainlit
- MCP tool support for external integrations
- Multi-LLM research with configurable LLM selection
- Automated web browsing with Playwright
- Token tracking and cost estimation
