## Overview

Project Alice is a framework and platform for building and deploying agentic workflows, supporting both code and no-code development approaches.

## Features

- No-code and coding interface for diverse developer needs
- Visual workflow builder for intuitive orchestration design
- Live demo available for easy evaluation
- Workflow deployment platform for production use
- Currently in alpha stage development
