## Overview

Parsee Datasets is a collection of datasets, case studies, and benchmarks designed for evaluating structured information extraction from various document formats. Created by the Parsee.ai team, it covers PDFs, HTML files, and images.

## Features

- Datasets for document information extraction
- Covers PDFs, HTML files, and images
- Benchmark case studies
- Evaluation metrics for extraction accuracy
- Supports document AI development

## Use Cases

- Benchmarking document AI extraction tools
- Evaluating information extraction from various formats
- Research on document understanding systems
