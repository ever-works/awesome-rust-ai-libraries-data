## Overview

ShampooSalesAgent is a minimal LLM sales agent framework designed for fast deployment and benchmarking. It supports multiple model providers including OpenAI, Claude, and HuggingFace models for sales automation tasks.

## Features

- Minimal LLM sales agent framework
- Fast deployment capabilities
- Supports OpenAI, Claude, and HuggingFace models
- Built-in benchmarking support
- Sales automation workflows

## Use Cases

- Rapid sales agent deployment
- Benchmarking sales AI agents
- Multi-model agent comparison
