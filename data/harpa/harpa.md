## Overview

Harpa is a Chrome extension that brings AI capabilities directly into the browser workflow for summarization, automation, and productivity.

## Features

- Content summarization
- Workflow automation
- AI-powered productivity tools
- GPT-4 integration
- Claude 2 integration
- Used by 300,000+ professionals

## Use Cases

- Web content summarization
- Browser-based task automation
- Research assistance
- Productivity enhancement
