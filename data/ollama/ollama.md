## Overview

Ollama is a tool for running large language models locally, offering easy setup across multiple platforms including macOS, Windows, Linux, and Docker.

## Features

- Easy local LLM setup and execution
- Multi-platform support: macOS, Windows, Linux, Docker
- Built-in library of pre-configured models
- Quickstart guides for customization and integration
- REST API for programmatic access
- Supports popular models like Llama, Mistral, Phi, and more

## Pricing

Free and open-source.
