## Overview

SkillLite is a lightweight, zero-dependency runtime for the agentsskills protocol, designed to enable AI agents to securely execute portable skills locally on end-user machines.

## Features

- Written in Rust for performance and safety
- Native OS sandboxing support for secure skill execution
- Millisecond cold starts
- Single binary deployment for easy distribution
- Enables portable skills that can be shared across different AI agent platforms
- Zero runtime dependencies
