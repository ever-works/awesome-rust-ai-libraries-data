## Overview

Impact Academy is a meta-benchmark that measures the ability of LLM agents to improve other LLM agents automatically. It evaluates meta-learning capabilities and agent self-improvement mechanisms.

## Features

- Meta-benchmark for agent improvement evaluation
- Tests LLM agents' ability to enhance other agents
- Auto-enhancement methodology
- Measures meta-learning capabilities
- Evaluates agent self-improvement strategies

## Use Cases

- Benchmarking agent self-improvement
- Evaluating meta-learning in LLM agents
- Research on automated agent enhancement
