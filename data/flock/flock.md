## Overview

Flock is a declarative multi-agent orchestration framework using blackboard architecture patterns and typed contracts for reliable agent coordination.

## Features

- Blackboard coordination pattern for agent collaboration
- Easily orchestrate complex patterns like multi-output fan-out
- Fluent DX enabling complex flows in a few lines of code
- Agents adhere to strict contracts instead of brittle natural language prompts
- Production-grade observability and zero-trust security model
