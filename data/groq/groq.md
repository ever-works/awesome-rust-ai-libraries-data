## Overview

Groq provides high-performance inference API endpoints with native tool use capabilities, enabling structured function calling from language models.

## Supported Models for Tool Use

- Llama 3 70B
- Llama 3 8B
- Mixtral 8x7B
- Gemma 7B IT

## Features

- Structured JSON output for function invocation
- Parallel tool calling support (Llama 3 models)
- High-speed inference hardware
- Programmatic operation execution
- Direct function invocation from model outputs
