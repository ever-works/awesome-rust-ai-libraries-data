## Overview

Guardrails is a Python framework focused on building reliable and safe AI applications through input/output validation and structured data generation.

## Features

- Input Guards for validating prompts and requests
- Output Guards for detecting and mitigating generated content risks
- Structured data generation from LLMs
- Risk detection and mitigation
- Validation pipelines

## Use Cases

- AI application safety
- Content filtering and moderation
- Structured output enforcement
- Prompt injection prevention
