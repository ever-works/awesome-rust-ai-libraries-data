## Overview

WebLINX is a benchmark for building and evaluating web navigation agents with conversational capabilities. It tests agents' ability to navigate websites and complete tasks while maintaining natural language dialogue with users.

## Features

- Web navigation benchmark with conversational interfaces
- Evaluates natural language interaction during navigation
- Tests web page understanding and interaction
- Combines navigation and dialogue capabilities
- Standardized evaluation for conversational web agents

## Use Cases

- Benchmarking conversational web navigation agents
- Evaluating agents that combine browsing and chat
- Research on interactive web agents
