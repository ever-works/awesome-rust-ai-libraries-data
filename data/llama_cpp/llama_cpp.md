## Overview

llama_cpp provides safe, high-level Rust bindings to llama.cpp, the widely-used C++ library for efficient LLaMA model inference. It wraps the native C++ API with Rust safety guarantees, making it easier to integrate LLaMA models into Rust code without manual memory management.

## Features

- Safe Rust bindings to llama.cpp C++ library
- High-level API abstracting C FFI complexity
- Full compatibility with GGUF/GGML model formats
- Local model inference on CPU with quantization support
