## Overview

Ungoliant is a data pipeline tool written in Rust for cleaning and preprocessing web text. It is used in the OSCAR corpus project to create high-quality, multilingual training datasets for language models from raw web data.

## Features

- Web text cleaning and preprocessing at scale
- Language identification and filtering
- Noise removal and deduplication
- Designed for LLM training dataset creation
- Rust-based for high throughput and low resource usage
