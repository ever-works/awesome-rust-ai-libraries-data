## Overview

Edge Transformers is a Rust wrapper around the ONNX Runtime that implements Hugging Face Optimum-style transformer pipelines. It is designed for edge and native inference, enabling deployment of transformer models in environments where Python is not available or practical.

## Features

- ONNX Runtime wrapper in Rust for transformer model inference
- Implements Hugging Face Optimum-style pipelines
- Designed for edge devices and native (non-Python) deployment
- Supports text classification, question answering, and other transformer tasks
