## Overview

CrewAI is a framework designed for orchestrating role-playing AI agents. It enables developers to build teams of agents that work together on complex tasks.

## Features

- Role-based agent design allowing specialization within agent teams
- Multi-agent collaboration for parallel task execution
- Flexible memory system for context retention across interactions
- Built-in error handling for robust agent workflows
