## Overview

AgentWatch is a multi-agent observability library designed to provide fleet-level monitoring that operates above per-agent tracing tools.

## Features

- Cascade failure detection across agent networks
- Heartbeat-based liveness monitoring for agents
- Cross-agent correlation for understanding inter-agent dependencies
- Forensic replay for debugging and analysis
- Fleet-level monitoring dashboard
- Multi-agent system observability

## Pricing

Open-source and free to use.
