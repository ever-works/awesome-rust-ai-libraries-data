## Overview

az-openai-rs is a Rust client for Azure's OpenAI Service. It provides compatibility with OpenAI API endpoints while using Azure-specific authentication methods (Azure endpoints and keys), enabling seamless integration with Azure infrastructure.

## Features

- Compatible with OpenAI API via Azure endpoints
- Azure-specific authentication (Azure keys and endpoints)
- Access to OpenAI models through Azure's managed service
- Integration with Azure infrastructure and security
