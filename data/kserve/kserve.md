## Overview

KServe is a highly scalable, standards-based Model Inference Platform on Kubernetes. It supports many ML frameworks including TensorFlow, PyTorch, ONNX, and XGBoost.

## Features

- Serverless inference with scale-to-zero
- Support for multiple ML frameworks
- Inference graph for multi-model composition
- Canary and A/B testing deployments
- Auto-scaling based on requests and concurrency
- Model storage and versioning
