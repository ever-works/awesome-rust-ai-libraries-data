## Overview

SafeAgentBench is a benchmark designed for evaluating the safety of task planning performed by embodied LLM agents. It tests agents' ability to execute tasks in physical or simulated environments while maintaining safety constraints and avoiding hazardous actions.

## Features

- Evaluates safe task planning for embodied agents
- Tests safety constraint adherence
- Assesses hazard avoidance in agent behaviors
- Standardized benchmarking for embodied agent safety
- Supports research on safe robotics and agent behaviors

## Use Cases

- Benchmarking embodied agent safety
- Evaluating safe task planning in agents
- Research on safety-aware agent design
