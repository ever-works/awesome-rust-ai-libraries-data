## Overview

LangGraph is a Python library that facilitates the construction of stateful, multi-actor LLM applications through cyclic graph-based orchestration.

## Features

- Stateful application development
- Multi-actor coordination
- Cyclic computation graphs
- Agent-like behavior patterns
- Streaming support
- Comprehensive guides and examples

## Architecture

- Graph-based workflow orchestration
- State persistence
- Multi-actor coordination
- Conditional edge routing
