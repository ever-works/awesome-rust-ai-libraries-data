## Overview

Langfuse is an open-source platform for engineering and improving LLM applications through debugging, monitoring, and prompt management.

## Features

- LLM application debugging
- Prompt management tools
- Performance metrics and analytics
- Open-source architecture
- Product Hunt #1 Golden Kitty winner in AI Infra

## Use Cases

- LLM application monitoring
- Prompt optimization
- Application debugging and testing
- Performance tracking
