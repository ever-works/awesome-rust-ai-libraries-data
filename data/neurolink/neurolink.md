## Overview

Neurolink is a multi-agent framework that unifies 12+ AI providers including OpenAI, Google, Anthropic, AWS, Azure, Groq, Together AI, Mistral, Cohere, Fireworks, Cloudflare, and Ollama into a single platform.

## Features

- Multi-agent framework with workflow orchestration capabilities
- Unified interface for 12+ AI providers across cloud and local
- Edge-first architecture supporting both local and cloud deployment
- Production-grade streaming and tool calling
- Battle-tested at Juspay processing 15M+ requests per month
