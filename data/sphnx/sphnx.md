## Overview

SPHNX is a modular benchmark suite designed to evaluate and enhance the privacy management capabilities of LLM-based agents. It tests agents' ability to handle sensitive data, respect privacy constraints, and comply with data protection requirements.

## Features

- Modular benchmark suite for privacy evaluation
- Tests LLM agent privacy management
- Evaluates data handling capabilities
- Assesses privacy compliance behaviors
- Modular architecture for extensible testing

## Use Cases

- Benchmarking privacy-aware LLM agents
- Evaluating data handling in agents
- Research on privacy-preserving agent design
