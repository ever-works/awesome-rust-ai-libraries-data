## Overview

LLMAgentOODGYM is an out-of-distribution benchmark study for LLM agents built on top of BrowserGym and AgentLab frameworks. It tests how well agents generalize to scenarios and environments not seen during training.

## Features

- Out-of-distribution evaluation for LLM agents
- Built on BrowserGym and AgentLab frameworks
- Tests agent generalization to unseen scenarios
- From ServiceNow Research
- Standardized OOD benchmarking methodology

## Use Cases

- Evaluating LLM agent generalization
- Benchmarking agent robustness to novel scenarios
- Research on agent out-of-distribution performance
