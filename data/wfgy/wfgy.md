## Overview

WFGY is an open-source framework designed for debugging and stress testing Large Language Models (LLMs) under long-horizon, high-tension text scenarios.

## Features

- Debugging framework for long-horizon text scenarios
- Stress testing capabilities for LLM reasoning
- TXT-based debugging application
- Structured sequence analysis to identify where reasoning breaks
- Retrieval failure detection and analysis
- High-tension scenario simulation
- Open-source and extensible

## Pricing

Open-source and free to use.
