## Overview

Tambo is a React-first framework for building AI-powered applications with generative UI capabilities and Model Context Protocol (MCP) integration.

## Features

- Component library for AI-driven UI composition
- MCP (Model Context Protocol) integration for tool connectivity
- TypeScript-first with full type safety
- SSR compatible with Next.js, Remix, and other frameworks
