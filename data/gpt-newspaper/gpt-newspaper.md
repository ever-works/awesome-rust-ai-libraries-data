## Overview

GPT Newspaper is an autonomous agent system that generates personalized newspapers using AI, orchestrated through six specialized sub-agents.

## Architecture

- **Search Agent**: Finds relevant content based on user preferences
- **Curation Agent**: Selects and filters the most relevant articles
- **Writing Agent**: Composes articles in appropriate style
- **Design Agent**: Handles layout and visual presentation
- **Editor Agent**: Reviews and refines content quality
- **Publishing Agent**: Finalizes and publishes the newspaper

## Features

- Multi-agent coordination for newspaper production
- Personalized content based on user preferences
- Automated end-to-end workflow
- Modular sub-agent architecture
