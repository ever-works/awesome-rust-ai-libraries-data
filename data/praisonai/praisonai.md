## Overview

PraisonAI is a production-ready multi-AI agents framework featuring self-reflection capabilities and support for over 100 LLM providers through LiteLLM.

## Features

- 100+ LLM support via LiteLLM integration
- MCP Protocol integration for tool extensibility
- Agentic workflows supporting route, parallel, loop, and repeat patterns
- Built-in memory with short-term, long-term, and entity memory
- Self-reflection for improved response quality
- Both Python and JavaScript SDKs available
