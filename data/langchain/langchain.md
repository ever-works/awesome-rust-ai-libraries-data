## Overview

Langchain is one of the most popular frameworks for building applications with large language models. It provides composability and modularity for constructing complex LLM-powered applications.

## Features

- Modular and extensible architecture for flexible application design
- Unified interface for interacting with multiple LLM providers
- Pre-built agent toolkits for common use cases
- CSV, JSON, and SQL agents for data-focused workflows
- Python and Pandas integration for data analysis
- Vector store capabilities for semantic search and RAG
