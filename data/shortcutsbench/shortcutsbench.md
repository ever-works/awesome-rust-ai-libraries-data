## Overview

ShortcutsBench is a large-scale, real-world benchmark designed for evaluating API-based agents. It assesses agents' ability to understand, navigate, and effectively utilize application programming interfaces to accomplish real-world tasks.

## Features

- Large-scale benchmark for API-based agents
- Real-world task scenarios
- Evaluates API navigation and usage capabilities
- Tests agent understanding of API documentation
- Comprehensive assessment of API-driven workflows

## Use Cases

- Benchmarking API-based agent capabilities
- Evaluating agents on real-world API tasks
- Research on API understanding and navigation
