## Overview

Transformers.js brings HuggingFace Transformers to the browser, enabling state-of-the-art ML inference directly on the client side. It supports text generation, embeddings, image classification, audio processing, and more.

## Features

- Runs entirely in the browser without a server
- Supports text generation, text classification, token classification
- Image classification and object detection
- Audio classification and speech recognition
- Embedding generation and similarity search
- Zero configuration API
- ONNX Runtime Web backend for inference acceleration
- Pre-trained models from HuggingFace Hub
