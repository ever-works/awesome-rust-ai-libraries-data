## Overview

Autodiff is a general-purpose automatic differentiation library for Rust, implementing both forward-mode and reverse-mode AD. It provides the core mathematical primitives needed for gradient-based optimization in machine learning.

## Features

- Forward-mode automatic differentiation
- Reverse-mode automatic differentiation (backpropagation)
- General-purpose AD interface usable in ML and scientific computing
- Chainable differentiation operations
