## Overview

GoodAI's Long Term Memory (LTM) Benchmark is a library designed to benchmark the long-term memory and continual learning capabilities of LLM-based agents. It provides comprehensive tests and code for evaluating how well agents retain and utilize information over extended interactions.

## Features

- Benchmarks long-term memory capabilities of LLM agents
- Tests continual learning abilities
- Standardized test suite with all tests and code included
- Evaluates memory retention across extended sessions
- Open-source for community contributions

## Use Cases

- Benchmarking agent memory systems
- Evaluating continual learning in LLM agents
- Research on long-term memory architectures
