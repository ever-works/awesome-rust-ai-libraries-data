## Overview

Agent OS provides a safety-first governance kernel for autonomous AI agents with POSIX-inspired resource management primitives.

## Features

- Policy engine with violation guarantees for safety
- Resource quotas and audit logging for agent oversight
- Integrations with CrewAI, LangChain, AutoGen, and Semantic Kernel
- MCP server support for protocol connectivity
