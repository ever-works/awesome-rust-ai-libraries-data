## Overview

RAG-Toolchain is a comprehensive Rust toolkit for building Retrieval-Augmented Generation workflows. It handles the full pipeline from document ingestion through indexing to query processing, enabling knowledge-grounded LLM applications.

## Features

- Document ingestion pipelines
- Indexing for efficient retrieval
- Query processing for LLM integration
- End-to-end RAG workflow support
