## Overview

RAI is an agentic framework designed for robotics applications, integrating AI agents with ROS 2 for intelligent robot control and interaction.

## Features

- ROS 2 integration for robotics middleware compatibility
- Complex action execution for sophisticated robot behaviors
- Scenario-based testing for validation and safety
- Voice interaction support for natural language robot control
- Vendor-agnostic architecture for hardware flexibility
