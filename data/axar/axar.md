## Overview

Axar is a minimal TypeScript agentic framework designed for building production-ready LLM applications with first-class TypeScript support and structured validation.

## Features

- Strongly-typed agent architecture with first-class TypeScript support
- Decorator-based API for defining agents, tools, and workflows
- Built-in structured I/O and validation powered by Zod
- Lightweight, production-oriented design
- Easy integration with OpenAI, Anthropic, Gemini, and other models
