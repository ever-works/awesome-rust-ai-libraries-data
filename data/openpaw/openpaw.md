## Overview

OpenPaw is an open-source CLI tool (npx pawmode) that turns Claude Code into a personal assistant with comprehensive built-in skills.

## Features

- 38 built-in skills: email, calendar, Spotify, smart home, Slack, GitHub, Telegram, Discord, and more
- No daemon required
- No cloud dependency
- Runs entirely locally
- MIT licensed
- Simple CLI interface
