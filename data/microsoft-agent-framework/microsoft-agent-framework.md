## Overview

Microsoft Agent Framework is a unified open-source framework combining AutoGen and Semantic Kernel for building AI agents and multi-agent workflows.

## Features

- Unified framework combining AutoGen and Semantic Kernel
- Multi-agent workflow orchestration
- LLM support
- Graph-based workflows for complex multi-step tasks
- Support for .NET and Python
- Open-source from Microsoft

## Pricing

Free and open-source.
