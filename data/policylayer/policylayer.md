## Overview

PolicyLayer provides non-custodial spending controls specifically designed for AI agents using crypto wallets. It enforces security policies without requiring custody of private keys.

## Features

- Daily spending limits
- Per-transaction caps
- Recipient whitelists
- Rate limiting
- Prevents wallet drains from bugs, prompt injection, or infinite loops
- Non-custodial — private keys never held by the service
