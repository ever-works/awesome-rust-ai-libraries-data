## Overview

Autonomous-GPT is a framework for building autonomous AI agents powered by GPT models that can operate independently to achieve defined goals.

## Features

- Internet access capabilities for real-time information retrieval
- Long-term memory for persistent context across sessions
- Goal-oriented behavior for directed autonomous operation
- File operations for reading and writing files
- Command execution for system-level task automation
