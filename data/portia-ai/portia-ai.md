## Overview

Portia AI is an open-source framework focused on building predictable, controllable, and authenticated AI agents for enterprise use cases.

## Features

- Structured planning for deterministic agent behavior
- Stateful execution tracking for complete auditability
- Human in the loop controls for critical decision points
- Tool catalogue with built-in authentication
- MCP support for extensibility
