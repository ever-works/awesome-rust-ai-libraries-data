## Overview

BentoML is an open-source model serving framework that simplifies the deployment of AI models and applications. It provides tools for building production-ready inference APIs, managing model versions, and scaling deployments.

## Features

- Build Model Inference APIs quickly
- Job queue support for async processing
- LLM application serving
- Multi-model pipeline orchestration
- Adaptive batching and traffic management
- Cloud-native deployment with Kubernetes and Docker
- Integration with major ML frameworks
- Built-in observability and monitoring

## Pricing

Free open-source core with BentoCloud enterprise offering.
