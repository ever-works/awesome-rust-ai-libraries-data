## Overview

LLM Agent Ask-For-Help is a benchmark that evaluates LLM agents' ability to recognize when they cannot complete sequential tasks and should quit or ask for help. It tests meta-cognitive self-awareness in agents.

## Features

- Evaluates agent self-awareness of limitations
- Sequential task evaluation
- Tests help-seeking behavior
- Meta-cognitive capability assessment
- Measures appropriate task abandonment

## Use Cases

- Benchmarking agent self-awareness
- Evaluating when agents should quit tasks
- Research on meta-cognitive agent capabilities
