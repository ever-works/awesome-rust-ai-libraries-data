## Overview

NeuroLink is a TypeScript-first agent framework designed for production deployments at enterprise scale.

## Features

- Multi-step agentic loops
- Tool execution control
- Persistent memory support (Redis, SQLite, S3)
- Human-in-the-loop (HITL) workflows
- MCP (Model Context Protocol) client integration
- Support for 13 LLM providers
- Production-proven at enterprise scale
- TypeScript-first development
