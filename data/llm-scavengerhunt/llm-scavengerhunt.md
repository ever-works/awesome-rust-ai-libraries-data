## Overview

LLM ScavengerHunt is a benchmark developed at UC Berkeley for evaluating Large Language Model agent tasks. It uses scavenger hunt scenarios to test multi-step reasoning, information gathering, and goal-directed behavior in agents.

## Features

- Scavenger hunt scenarios for agent evaluation
- Multi-step reasoning assessment
- Goal-directed behavior testing
- From UC Berkeley
- Information gathering evaluation

## Use Cases

- Benchmarking LLM agent capabilities
- Evaluating multi-step reasoning in agents
- Research on goal-directed agent behavior
