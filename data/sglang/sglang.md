## Overview

SGLang is a fast serving framework designed for both large language models and vision language models. It provides optimized inference with support for structured generation and multi-modal inputs.

## Features

- Fast serving for LLMs and VLMs
- Structured generation with JSON schema guidance
- Optimized RadixAttention for KV cache management
- Support for multiple backends including tensor parallelism
- Native multi-modal processing support
- OpenAI-compatible API server
