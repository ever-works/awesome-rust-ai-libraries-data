## Overview

hf-hub is a Rust client for the Hugging Face Hub, enabling programmatic downloads of models and datasets. It mirrors many features of the Python huggingface_hub library, bridging the gap between Rust and the Hugging Face ecosystem.

## Features

- Download models from the Hugging Face Hub
- Download datasets from the Hub
- Mirrors Python huggingface_hub functionality
- Authentication and repository management support
