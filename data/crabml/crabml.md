## Overview

crabml is a fast, cross-platform AI inference engine written in Rust that leverages WebGPU for hardware acceleration, providing efficient LLM inference with features like memory-mapped model loading and quantization support.

## Features

- Fast cross-platform AI inference
- LLM inference support
- Memory-mapped (mmap) model loading for zero-copy access
- Quantization support for reduced memory footprint
- WebGPU-based hardware acceleration

## Backend Support

- CPU: Host-side computation
- WGPU: WebGPU-based cross-platform GPU acceleration

## Dependencies

- vulkano: Vulkan bindings
- wgpu: WebGPU implementation
