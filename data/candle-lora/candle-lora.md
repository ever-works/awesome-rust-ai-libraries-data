## Overview

candle-lora is a Rust library offering efficient LoRA (Low-Rank Adaptation) support for Candle models.

## Features

- Out-of-the-box LoRA support for numerous Candle models (see [examples](https://github.com/EricLBuehler/candle-lora/tree/master/candle-lora-transformers/examples)).

## Pricing

Free and open-source.
