## Overview

GenoMAS is a multi-agent framework specifically designed for scientific discovery workflows and automated data analysis with a domain-agnostic approach.

## Features

- Guided planning framework for structured scientific workflows
- Typed message-passing protocol for reliable inter-agent communication
- Heterogeneous LLM architecture for diverse analytical tasks
- Domain-agnostic design applicable across scientific fields
- Scientific workflow automation for reproducible research
