## Overview

CrustAGI is a Rust port of the BabyAGI architecture, an autonomous task management agent. It uses GPT models to create, prioritize, and execute tasks iteratively, demonstrating agentic behavior in Rust.

## Features

- Autonomous task management agent
- Iterative task creation, prioritization, and execution
- Uses GPT for task planning and decision-making
- Rust port of the BabyAGI concept
