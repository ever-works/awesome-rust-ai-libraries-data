## Overview

A versatile answer engine combining multiple LLM providers and search APIs to deliver sophisticated responses with citation-backed queries.

## Features

- Leverages Groq, Mistral AI, Langchain.JS, Brave Search, Serper API, and OpenAI
- RAG for citation-backed search queries
- Reduced hallucination through source citations
- Efficient answer delivery
- Multi-provider integration

## Pricing

Free and open-source.
