## Overview

Entroly is an Information-theoretic Context Engineering Engine designed to optimize the selection and pruning of RAG (Retrieval-Augmented Generation) fragments for LLM applications.

## Features

- Uses reinforcement learning to intelligently prune and select optimal RAG fragments
- Information-theoretic approach to context engineering
- Optimizes context window usage for large language models
- Helps improve the quality and relevance of retrieved context fragments
