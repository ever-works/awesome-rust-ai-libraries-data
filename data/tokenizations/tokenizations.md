## Overview

Tokenizations provides alignment between tokenizer output and original text. It maps token indices back to their source substrings, which is essential for tasks like named entity recognition, span extraction, and text highlighting.

## Features

- Aligns tokenized output with original source text
- Maps token indices to character-level substrings
- Useful for NER, span extraction, and text highlighting
- Supports multiple tokenizer formats
