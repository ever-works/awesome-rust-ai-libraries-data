## Overview

AIP (Agent Identity Protocol) is a decentralized identity system designed for AI agents, providing cryptographic verification mechanisms and trust-based communication networks.

## Features

- Cryptographic Ed25519/DID identity for agents
- Vouch-based trust graphs
- End-to-end encrypted messaging
- CLI for agent identity management
- Python SDK via PyPI (`pip install aip-identity`)

## Pricing

Free and open-source.
