## Overview

LLM Dataset Converter is a library for converting datasets between different formats used in LLM training. It provides tools to transform existing datasets into compatible formats for various model training pipelines.

## Features

- Multi-format dataset conversion
- Compatible with various training pipelines
- Streamlined data preprocessing
- Supports multiple output formats

## Use Cases

- Converting datasets for LLM training
- Format normalization for model pipelines
- Dataset preparation workflows
