## Overview

Widemem AI is a lightweight Python memory layer designed for LLM agents, providing structured memory management with local-first architecture.

## Features

- Importance scoring for memory entries
- Temporal decay to manage information relevance over time
- 3-tier hierarchical memory organization
- YMYL (Your Money or Your Life) prioritization for critical information
- Batch conflict resolution for handling contradictory memories
- Local-first architecture using SQLite + FAISS
- Lightweight and easy to integrate
