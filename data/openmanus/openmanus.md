## Overview

OpenManus is an open-source AI agent platform built by MetaGPT contributors, designed for building general-purpose agents with modular customization.

## Features

- Modular architecture for customization and extensibility
- Data analysis and visualization agents for analytical workflows
- Reinforcement learning integration (OpenManus-RL)
- No invitation code required — fully open access
- Built by MetaGPT contributors with proven agent development experience
