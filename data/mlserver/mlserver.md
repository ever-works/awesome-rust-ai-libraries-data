## Overview

MLServer is a flexible, multi-framework inference server designed for ML models. It provides a standardized way to serve models in production environments.

## Features

- Support for multiple ML frameworks
- Multi-model serving capability
- Standardized REST and gRPC APIs
- Kubernetes-native deployment
- Prometheus metrics integration
- Pluggable runtime architecture
