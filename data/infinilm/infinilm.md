## Overview

InfiniLM is a transformer model inference project developed from YdrMaster/llama2.rs, focusing on large language model inference with broad multi-hardware support including CPU, CUDA, OpenCL, and Ascend NPU.

## Features

- LLM inference for transformer-based models
- Multiple hardware backend support
- Derived from llama2.rs project
- Operators library for cross-platform computation

## Backend Support

- CPU: Host-side computation
- CUDA: NVIDIA GPU acceleration
- OpenCL: Cross-platform GPU acceleration
- Ascend: Huawei Ascend NPU support

## Dependencies

- operators: Multi-hardware operator library
