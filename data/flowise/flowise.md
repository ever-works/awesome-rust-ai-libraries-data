## Overview

Flowise is a drag-and-drop UI framework that simplifies building LLM-powered applications through a visual interface.

## Features

- Visual flow builder for intuitive workflow design
- Custom LLM integrations for various model providers
- API generation for exposing flows as REST endpoints
- Authentication support for secure deployments
- Docker deployment for containerized production use
