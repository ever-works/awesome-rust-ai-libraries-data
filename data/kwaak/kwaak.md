## Overview

Kwaak is an open-source application from Bosun AI that orchestrates multiple AI coding agents running in parallel on a codebase. Built on Swiftide, it leverages Rust's concurrency capabilities to coordinate multiple agents simultaneously.

## Features

- Runs multiple AI coding agents in parallel
- Code analysis and modification workflows
- Built on the Swiftide framework
- Open-source implementation
- Concurrent agent orchestration
