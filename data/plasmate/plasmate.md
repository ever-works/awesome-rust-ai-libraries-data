## Overview

Plasmate is a browser engine purpose-built for AI agents, designed to optimize how agents interact with web content.

## Features

- Compiles HTML into Semantic Object Model (SOM) format
- 10x token compression compared to raw HTML
- V8 JavaScript rendering engine
- Chrome DevTools Protocol (CDP) compatibility
- Authenticated browsing support
- Integrated MCP server for agent communication
- Optimized for AI agent web navigation and interaction

## Pricing

Open-source and free to use.
