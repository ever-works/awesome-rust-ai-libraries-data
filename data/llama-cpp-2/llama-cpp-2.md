## Overview

llama-cpp-2 is a low-level Rust binding that mirrors the llama.cpp C API as closely as possible. Unlike higher-level wrappers, it provides a thin layer over the raw C interface, giving developers direct control over llama.cpp's full feature set.

## Features

- Thin Rust wrapper closely mirroring llama.cpp C API
- Direct access to all llama.cpp functions and data structures
- Minimal abstraction overhead for maximum control
- Suitable for building custom higher-level abstractions on top
