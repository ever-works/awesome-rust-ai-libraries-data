## Overview

IvyCheck provides real-time safety checking APIs for AI applications, specifically designed to protect agentic AI systems from common vulnerabilities.

## Features

- Prompt injection attack prevention
- PII data leakage detection
- Hallucination detection
- Real-time safety checks
- Designed for agentic AI development

## Use Cases

- Agentic AI security
- Production AI application safety
- Data loss prevention
- Output validation
