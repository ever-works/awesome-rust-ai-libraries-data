## Overview

openai-safe focuses on type safety for OpenAI API interactions in Rust. It uses strongly typed request and response structures to catch errors at compile time rather than runtime, reducing the risk of API misuse.

## Features

- Strongly typed API request and response structures
- Compile-time validation of API parameters
- Reduces runtime errors through Rust's type system
- Supports OpenAI completions, chat, and embeddings
