## Overview

The AI Researcher is an AI agent that leverages Claude 3 and SERPAPI for in-depth topic research, refining subtopic analyses into comprehensive reports.

## Features

- Claude 3 integration for analysis
- SERPAPI for web search
- Subtopic refinement and analysis
- Comprehensive report generation
- Customizable research parameters

## Requirements

Requires API keys for Claude 3 and SERPAPI functionality.
