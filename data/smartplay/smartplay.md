## Overview

SmartPlay is a benchmark for Large Language Models that uses a variety of games to test various important LLM capabilities as agents. It evaluates planning, reasoning, sequential decision-making, and interaction abilities through game-based tasks.

## Features

- Game-based benchmark for LLM agents
- Tests planning and reasoning capabilities
- Evaluates sequential decision-making
- Multiple game environments for diverse testing
- Quantitative metrics for agent capabilities

## Use Cases

- Benchmarking LLM agent capabilities through games
- Evaluating planning and reasoning in agents
- Research on game-based agent evaluation
