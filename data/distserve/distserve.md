## Overview

DistServe is a disaggregated LLM serving system that separates the prefill and decode phases across different servers, optimizing resource allocation and improving overall throughput.

## Features

- Separates prefill and decode computation phases
- Optimizes resource allocation across servers
- Improved throughput for LLM serving
- Handles variable request patterns efficiently
- Disaggregated architecture for scalability
