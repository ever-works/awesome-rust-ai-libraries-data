## Overview

Brokenhill is a productionized Greedy Coordinate Gradient (GCG) attack tool for Large Language Models. It implements adversarial attacks that find input suffixes to bypass safety filters and test model robustness.

## Features

- Greedy Coordinate Gradient (GCG) attacks
- Automated adversarial suffix generation
- Production-grade implementation
- Support for multiple model architectures
- Evaluates model vulnerability to prompt injection
