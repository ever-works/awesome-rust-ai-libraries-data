## Overview

ndarray is a foundational library for handling n-dimensional arrays in Rust, originally developed by bluss.

## Features

- Array views for zero-copy subarrays
- Multidimensional slicing and indexing
- Efficient vectorized operations
- Broadcasting support
- Parallel iterator support via Rayon

## Use Cases

- Numerical simulations
- Image processing
- Machine learning tensor operations

Free and open-source.
