## Overview

CoreAgent is a minimalist agent framework emphasizing simplicity while providing stateful tool management and multi-agent coordination.

## Features

- Simplicity-first design for low barrier to entry
- Stateful tool support for persistent tool context
- Multi-agent coordination for collaborative workflows
- Shared state management across agents
- Built-in tools library for common operations

- Language: Python
