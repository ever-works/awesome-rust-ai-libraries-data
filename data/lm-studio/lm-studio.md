## Overview

LM Studio provides a user-friendly platform for running local language models offline across multiple platforms.

## Features

- Support for multiple local LLMs (LLaMa, Falcon, MPT, etc.)
- Chat UI for model interaction
- OpenAI-compatible server
- Hugging Face model downloads
- Cross-platform support (Mac, Windows, Linux)
- Privacy-focused with no data collection
- Free for personal use
