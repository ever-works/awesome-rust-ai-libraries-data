## Overview

Kiln AI provides a suite of tools for building AI products, ranging from agents and evaluation frameworks to RAG systems and fine-tuning datasets.

## Features

- Free desktop application for no-code AI development
- Evaluation frameworks for measuring AI quality
- RAG system building tools for retrieval-augmented generation
- Synthetic data generation for training and testing
- Agent development tools for building custom AI agents
