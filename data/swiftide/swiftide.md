## Overview

Swiftide is a Rust library for building Retrieval-Augmented Generation (RAG) pipelines and agents. It provides a complete toolkit for ingesting data, creating vector indices, and managing prompt injection workflows.

## Features

- Fast data ingestion pipelines
- Vector indexing for embedding storage
- Prompt injection and management
- Agent building capabilities
- Optimized for Rust performance characteristics
