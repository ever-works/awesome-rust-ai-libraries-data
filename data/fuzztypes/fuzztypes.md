## Overview

FuzzTypes is a Pydantic extension library designed to enhance data conversion capabilities for AI agents through autocorrecting annotation types.

## Features

- Extends Pydantic's data conversions with autocorrecting annotation types
- Enables named entity linking for normalization
- Transforms raw string data into structured, intelligent entities
- Integrates seamlessly with existing Pydantic-based workflows
- Enhances data quality for AI agent inputs and outputs

## Use Cases

- Structured data normalization in AI agent pipelines
- Named entity recognition and linking
- Data validation and transformation for LLM applications
