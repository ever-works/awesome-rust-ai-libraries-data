## Overview

Cordum is a safety-first agent orchestration platform designed to ensure secure agent execution through pre-dispatch policy evaluation and output quarantining.

## Features

- Pre-dispatch safety policy evaluation before agent execution
- Output scanning and quarantine for dangerous content
- Job scheduling and workflow engine for orchestration
- MCP server integration for tool connectivity
- gRPC safety kernel for secure communication
