## Overview

The Open Assistant API is a self-hosted, open-source framework that enables the creation of customized AI assistants with orchestration and extension capabilities.

## Features

- Self-hosted deployment
- Compatible with OpenAI's Assistants API
- Integration with OpenAI's LLM and LangChain SDK
- Seamless orchestration and extension capabilities
- Customizable AI assistant creation

## Pricing

Free and open-source.
