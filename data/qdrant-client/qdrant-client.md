## Overview

The Qdrant-Client is the official Rust client for the Qdrant vector database. It enables efficient storage, indexing, and querying of embedding vectors, supporting semantic search and recommendation systems in Rust applications.

## Features

- Full Rust client for Qdrant vector database
- Store and query embedding vectors efficiently
- Vector similarity search (cosine, Euclidean, dot product)
- Payload filtering and metadata management
- CRUD operations for collections and points
- Async API support
