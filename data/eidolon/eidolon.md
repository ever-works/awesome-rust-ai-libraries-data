## Overview

Eidolon is the first AI Agent Server, providing a pluggable Agent SDK and enterprise-ready deployment server for agentic applications. It takes a server-first approach to building and hosting AI agents.

## Features

- Pluggable agent SDK for building custom agents
- Enterprise-ready deployment server
- Server-first architecture for agentic applications
- Support for multiple LLM providers
- Agent lifecycle management
- Scalable agent orchestration

## Pricing

Free and open-source.
