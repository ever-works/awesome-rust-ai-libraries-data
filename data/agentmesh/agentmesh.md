## Overview

AgentMesh provides a secure trust layer for multi-agent ecosystems using zero-trust governance principles and cryptographic identity management.

## Features

- Ed25519 cryptographic agent identity for secure authentication
- Delegation chains with bounded depth for controlled authority
- HTTP trust middleware for Flask and FastAPI
- A2A protocol compatible for agent-to-agent communication
