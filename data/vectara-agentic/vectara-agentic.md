## Overview

Vectara-agentic is a framework for creating AI Assistants and agentic applications powered by Vectara's search infrastructure.

## Features

- AI assistant creation framework
- Agentic application support
- Built on Vectara search and retrieval
- Knowledge-driven agent reasoning
- Tool use integration
- Streamlined assistant deployment

## Pricing

Free and open-source framework; Vectara service may have separate pricing.
