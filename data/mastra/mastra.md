## Overview

Mastra is a TypeScript-native AI agent framework that provides type-safe development with built-in RAG, routing, and observability capabilities.

## Features

- Type-safe development with full TypeScript support
- Multi-model support and router (GPT-4, Claude, Gemini, Llama)
- Structured responses for reliable data handling
- API generation for exposing agents as services
- MCP generation for protocol-compatible tool integration
- Tool integration for extended agent capabilities
