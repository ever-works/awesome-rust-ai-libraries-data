## Overview

OpenAI Swarm is an educational framework managed by OpenAI's Solutions team for building ergonomic, lightweight multi-agent orchestration systems.

## Features

- Lightweight multi-agent orchestration
- Ergonomic API design
- Educational purpose with best practices
- Managed by OpenAI Solutions team
- Simple and intuitive interface
- Suitable for prototyping and learning

## Pricing

Free and open-source.
