## Overview

Oxidized-Transformers is a pure Rust implementation of Hugging Face-style transformer models. It eliminates the need for Python dependencies or C++ FFI bindings by implementing the entire transformer architecture natively in Rust.

## Features

- Pure Rust implementation with no external Python or C++ dependencies
- Experimental support for GPT and BERT model architectures
- Compatible with transformer model weights from the Hugging Face ecosystem
- Memory-safe inference leveraging Rust's ownership model
- Suitable for embedding transformer models into Rust-native applications

## Status

Experimental — actively developed but not yet ready for production use.
