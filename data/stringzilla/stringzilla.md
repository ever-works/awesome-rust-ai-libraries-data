## Overview

StringZilla is a SIMD-accelerated string library providing high-performance implementations of common string operations including search, sort, edit distances, alignments, and string generators. Optimized for x86 AVX2 & AVX-512 and Arm NEON architectures.

## Features

- SIMD-accelerated string search algorithms
- String sorting with hardware-optimized comparators
- Edit distance calculations (Levenshtein, etc.)
- Sequence alignments for bioinformatics and NLP
- String generators for combinatorial operations
- x86 AVX2 and AVX-512 instruction set support
- Arm NEON instruction set support
- Zero dependency design for easy integration

## Use Cases

- Text search and indexing in NLP pipelines
- Spell checking and fuzzy string matching
- DNA/RNA sequence alignment
- Log parsing and pattern matching
- Deduplication and similarity detection
