## Overview

Balrog is a benchmark designed for assessing agentic reasoning capabilities of Large Language Models (LLMs) and Vision-Language Models (VLMs) using game-based evaluation tasks. It tests agents' ability to reason, plan, and make decisions in complex, multi-step game environments.

## Features

- Game-based evaluation for LLM and VLM agents
- Tests multi-step reasoning capabilities
- Evaluates agentic decision-making in complex environments
- Supports both language and vision-language models
- Quantitative assessment of agent reasoning abilities

## Use Cases

- Benchmarking LLM reasoning through game tasks
- Comparing VLM agentic capabilities
- Research on multi-step reasoning evaluation
