## Overview

Elia is a keyboard-centric terminal user interface for interacting with large language models.

## Features

- Terminal-based interaction
- Multi-model support: ChatGPT, Claude, Llama 3, Phi 3, Mistral, Gemma
- Easy model switching
- Local model support
- Local SQLite database for conversation storage
- Efficient keyboard navigation

## Pricing

Free and open-source.
