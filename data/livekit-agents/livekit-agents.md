## Overview

LiveKit Agents is an open-source framework for building programmable participants in real-time WebRTC sessions.

## Features

- Real-time audio, video, and data stream processing
- Server-side participant implementation
- WebRTC session integration
- Audio and video generation
- Open-source framework

## Use Cases

- Real-time AI voice agents
- Video stream processing
- Interactive AI applications
- WebRTC-based AI services
