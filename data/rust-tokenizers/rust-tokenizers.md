## Overview

rust-tokenizers is a standalone Rust crate providing high-performance implementations of popular tokenizer algorithms used in NLP. It supports BERT WordPiece, SentencePiece, and other common tokenization schemes.

## Features

- Support for BERT WordPiece tokenization
- SentencePiece support for subword tokenization
- High-performance implementations optimized for speed
- Independent of the Hugging Face tokenizers library
- Suitable for building custom NLP pipelines in Rust
