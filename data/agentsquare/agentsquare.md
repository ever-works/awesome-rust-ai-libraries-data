## Overview

AgentSquare is a Tsinghua University research project providing automatic LLM agent search within a modular design space for optimizing agent architectures.

## Features

- Module evolution and recombination for agent optimization
- Planning, Reasoning, Tool Use, and Memory modules
- Adaptive agent search framework for optimal configurations
- Unified framework for six different agent tasks
- Tsinghua University research project with academic backing
