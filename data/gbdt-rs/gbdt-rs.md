# gbdt-rs

MesaTEE GBDT-RS is a fast and secure Gradient Boosting Decision Tree library, supporting Trusted Execution Environments (TEEs) such as Intel SGX and ARM TrustZone. Designed for privacy-preserving machine learning in secure enclaves.
