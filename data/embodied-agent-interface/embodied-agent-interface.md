## Overview

Embodied Agent Interface is a project and benchmark for evaluating LLMs in embodied decision-making scenarios. It provides evaluation frameworks for agents that need to interact with physical or simulated environments to accomplish tasks.

## Features

- Benchmarks LLMs for embodied decision making
- Evaluates physical environment interaction
- Task-based evaluation in simulated/real environments
- Standardized metrics for embodied agent performance

## Use Cases

- Benchmarking embodied AI agents
- Evaluating robotic agent capabilities
- Research on physical world agent interactions
