## Overview

Lorax (Low-Rank Adaptation eXchange) is a high-performance serving system designed to serve thousands of fine-tuned LLMs efficiently by sharing base model weights across multiple LoRA adapters.

## Features

- Serves up to 1000s of fine-tuned models simultaneously
- Dynamic LoRA loading and activation
- Shared base model weights
- Multi-adapter support with minimal overhead
- OpenAI-compatible API
- Integration with HuggingFace ecosystem
