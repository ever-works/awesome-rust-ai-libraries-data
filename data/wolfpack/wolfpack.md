## Overview

Wolfpack is a PWA command center for controlling AI coding agents like Claude, Codex, and Gemini across multiple machines. It uses tmux sessions for multi-terminal control, secured by Tailscale.

## Features

- Multi-terminal grid view for simultaneous agent control
- Mobile-optimized touch UI for on-the-go management
- Tailscale security for encrypted tunnel connections
- Ralph — built-in autonomous task runner
- Support for Claude, Codex, and Gemini
- Cross-machine orchestration capabilities
