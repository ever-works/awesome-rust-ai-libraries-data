## Overview

The Google Agent Development Kit (ADK) provides tools for building, evaluating, and deploying production-ready AI agents with a code-first approach.

## Features

- Code-first development approach for programmatic agent building
- Modular multi-agent systems for complex orchestrations
- Model-agnostic and deployment-agnostic design
- Built-in developer UI (adk-web) for visualization and debugging
- Deploy on Cloud Run or Vertex AI for scalable production
