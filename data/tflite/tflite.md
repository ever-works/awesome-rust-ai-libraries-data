## Overview

The `tflite` crate provides Rust bindings for TensorFlow Lite, Google's lightweight deep learning framework for mobile and embedded devices. It enables running pre-trained TensorFlow models on edge hardware with minimal latency and resource consumption.

## Features

- Direct bindings to the TensorFlow Lite C API
- Model loading and inference from .tflite files
- Support for various model operations and layers
- Designed for on-device machine learning deployment
- Idiomatic Rust interface over the underlying C library
