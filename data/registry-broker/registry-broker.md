## Overview

Registry Broker is a universal index and routing layer for AI agents that aggregates metadata from multiple registries across web2 and web3 ecosystems.

## Supported Registries

- NANDA
- MCP (Model Context Protocol)
- Virtuals
- OpenRouter
- A2A (Agent-to-Agent)
- X402 Bazaar

## Features

- Universal agent indexing across registries
- Agentic metadata aggregation
- Agent profile normalization
- Protocol translation between different agent ecosystems
- Cross-web2/web3 compatibility

Free and open-source.
