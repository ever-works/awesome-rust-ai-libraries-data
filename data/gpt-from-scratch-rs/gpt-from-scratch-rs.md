## Overview

gpt-from-scratch-rs is a Rust port of Andrej Karpathy's popular "Let's build GPT: from scratch, in code, spelled out" YouTube tutorial. It demonstrates how to build and train a GPT-style language model from the ground up using the Candle ML framework.

## Features

- Step-by-step GPT model implementation following Karpathy's tutorial
- Showcases the Candle API on a practical training problem
- Educational resource for understanding GPT architecture and training
- Built entirely in Rust using Candle as the underlying ML framework
- Includes text data tokenization and character-level language modeling

## Use Cases

- Learning GPT architecture fundamentals through hands-on implementation
- Understanding Candle's API and capabilities for model training
- Educational reference for building transformer-based models in Rust
