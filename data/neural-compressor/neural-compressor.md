## Overview

Intel Neural Compressor is an open-source library that provides state-of-the-art model compression techniques including quantization, pruning, and knowledge distillation. It supports low-bit quantization (INT8, FP8, INT4, FP4, NF4) for large language models.

## Features

- Multi-framework support: TensorFlow, PyTorch, ONNX Runtime
- INT8/FP8/INT4/FP4/NF4 quantization
- Automatic mixed precision and sparsity
- Built-in benchmarking tools
- Hardware-aware optimization for Intel and other platforms
- Model compression techniques including pruning and distillation
