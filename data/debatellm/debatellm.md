## Overview

DebateLLM is a benchmark that evaluates multi-agent debate between language models for assessing truthfulness in question and answer tasks. It investigates whether structured debate between LLMs can improve answer accuracy and truthfulness.

## Features

- Multi-agent debate framework for LLMs
- Truthfulness evaluation in Q&A
- Structured debate protocols
- Measures accuracy improvement through debate
- Benchmark for LLM truthfulness assessment

## Use Cases

- Evaluating truthfulness in LLM responses
- Testing multi-agent debate methodologies
- Research on self-correction through debate
