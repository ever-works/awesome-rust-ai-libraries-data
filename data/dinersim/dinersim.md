## Overview

DinerSim is a restaurant simulation benchmark designed to evaluate LLM-based multi-agent cooperation capabilities. It tests agents' ability to collaborate, coordinate roles, and manage tasks in a simulated restaurant environment.

## Features

- Restaurant simulation environment
- Multi-agent cooperation evaluation
- Collaborative task planning assessment
- Role-based agent coordination testing
- Standardized simulation benchmark

## Use Cases

- Benchmarking multi-agent cooperation
- Evaluating collaborative task planning
- Research on agent coordination strategies
