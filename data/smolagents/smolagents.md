## Overview

Smolagents by Hugging Face is a minimalist agent framework that emphasizes simplicity while providing powerful multi-agent orchestration and tool integration capabilities.

## Features

- Code-first approach for programmatic agent development
- Multi-agent orchestration for complex workflows
- LLM provider flexibility supporting various model backends
- Tool integration for extended agent capabilities
- Hub integration for sharing agents and tools
