## Overview

Sayna is an open-source voice infrastructure layer for AI agents providing unified APIs for voice capabilities.

## Features

- Unified APIs for STT/TTS
- Real-time audio streaming
- SIP Telephony support
- Multi-provider support: OpenAI, Deepgram, ElevenLabs, Google, Azure
- Self-hostable
- WebSocket/REST interfaces
- Voice-enabled agent development

## Pricing

Free and open-source.
