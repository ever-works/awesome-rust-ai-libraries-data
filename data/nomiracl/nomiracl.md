## Overview

NoMIRACL (Noisy Multilingual Information Retrieval Assertion Check List) is a multilingual hallucination evaluation dataset designed to evaluate LLM robustness in RAG pipelines when faced with first-stage retrieval errors. It covers 18 languages to assess cross-lingual RAG reliability.

## Features

- Multilingual hallucination evaluation
- Covers 18 languages
- Evaluates RAG robustness against retrieval errors
- First-stage retrieval error assessment
- Cross-lingual reliability testing

## Use Cases

- Evaluating RAG system robustness across languages
- Benchmarking hallucination rates in multilingual settings
- Research on retrieval-error resilience
