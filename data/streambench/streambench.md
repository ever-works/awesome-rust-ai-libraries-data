## Overview

StreamBench is a benchmark designed to evaluate LLM agents' ability to improve over time in streaming scenarios. It tests continuous learning and adaptation capabilities as agents process ongoing streams of tasks and information.

## Features

- Evaluates LLM agent improvement over time
- Streaming scenario benchmarking
- Tests continuous learning capabilities
- Assesses adaptation to evolving tasks
- Pioneering methodology for temporal agent evaluation

## Use Cases

- Benchmarking continuous learning in agents
- Evaluating agent improvement in streaming contexts
- Research on temporal agent capabilities
