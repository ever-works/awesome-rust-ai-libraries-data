## Overview

AgentFlow is a Stanford research project providing a trainable multi-agent framework with reinforcement learning optimization integrated into the workflow.

## Features

- Four specialized modules: planner, executor, verifier, generator
- Flow-GRPO reinforcement learning for in-the-flow optimization
- Tool integration across math, coding, scientific, and search domains
- Stanford research project with academic rigor
- Performance gains over monolithic agent approaches
