## Overview

AppWorld is a benchmark that won the ACL 2024 Best Resource Paper award. It provides a controllable world of apps and people for benchmarking interactive coding agents, offering realistic application interaction scenarios.

## Features

- Controllable simulated app environment for agent benchmarking
- Interactive coding agent evaluation
- Best Resource Paper at ACL 2024
- Simulated apps and user interactions
- Public leaderboard for agent performance comparison

## Use Cases

- Benchmarking interactive coding agents
- Evaluating agents on app interaction tasks
- Research on automated app usage
