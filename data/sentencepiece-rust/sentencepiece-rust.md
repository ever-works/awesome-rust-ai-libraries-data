## Overview

This crate provides Rust bindings to Google's SentencePiece tokenizer. SentencePiece is a subword segmentation tool that implements unigram language model segmentation, widely used for tokenization in many NLP models.

## Features

- Bindings to Google's SentencePiece C++ library
- Unigram language model segmentation
- Compatible with pretrained SentencePiece models
- Subword tokenization for multilingual text
