## Overview

Ort provides Rust bindings for Microsoft's ONNX Runtime, allowing Rust applications to load and execute ONNX-format machine learning models on both CPU and GPU. It supersedes the older onnxruntime-rs crate with a more complete and modern API.

## Features

- Bindings to Microsoft ONNX Runtime for model execution
- CPU and GPU inference support
- Supersedes older onnxruntime-rs with improved API design
- Supports all ONNX model operators and execution providers
- Compatible with models exported from PyTorch, TensorFlow, and other frameworks
