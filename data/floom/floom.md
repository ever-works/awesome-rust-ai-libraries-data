## Overview

Floom is an AI gateway and marketplace that enables streamlined integration of AI agents and pipelines using a Kubernetes-style architecture.

## Features

- AI pipeline orchestration for complex workflows
- Agent marketplace integration for discovery and deployment
- Kubernetes-style architecture for scalable deployments
- Streamlined AI integration for developer experience
- DevOps-friendly deployment for production operations
