## Overview

CallM is a high-level Rust crate that provides an easy-to-use API for running local LLMs on both CPUs and GPUs. It leverages the Candle ML framework as its backend, supporting models like LLaMA 2 and Mistral.

## Features

- High-level API abstracting away low-level model loading and inference details
- Supports LLaMA 2, Mistral, and other popular open-weight LLMs
- CPU and GPU inference via Candle backend
- Simple integration into Rust applications
