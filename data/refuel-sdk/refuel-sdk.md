## Overview

Refuel SDK is a tool for labeling, cleaning, and enriching text datasets using Large Language Models. It provides an automated pipeline for improving data quality, which is critical for training and evaluating ML models.

## Features

- Automated text dataset labeling with LLMs
- Data cleaning and deduplication
- Dataset enrichment pipelines
- Improves data quality for ML workflows
- Integrates with existing data processing pipelines

## Use Cases

- Cleaning and labeling training datasets
- Enriching existing text datasets
- Improving data quality for ML model training
