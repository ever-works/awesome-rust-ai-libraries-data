## Overview

GPT-Annotate is an easy-to-use Python package designed to streamline automated text annotation using LLMs. It supports different NLP tasks and dataset creation workflows.

## Features

- Automated text annotation using LLMs
- Easy-to-use Python package
- Supports multiple NLP annotation tasks
- Streamlines dataset labeling workflows
- Integrates with existing data pipelines

## Use Cases

- Automated dataset annotation
- NLP task labeling
- Accelerating dataset creation for ML models
