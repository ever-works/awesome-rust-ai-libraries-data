## Overview

Embedchain is a framework that simplifies building ChatGPT-like bots capable of ingesting data from multiple sources and generating contextual responses.

## Features

- Multi-source data ingestion from files, URLs, and APIs
- Automated embedding pipeline for seamless data preparation
- Context window management for optimal LLM performance
- Multiple LLM support for flexible model selection
- RAG optimization for improved retrieval accuracy
