## Overview

Petals is a decentralized framework for running LLMs collaboratively across multiple machines, similar to BitTorrent. It enables users to run inference and fine-tune large models even on consumer-grade hardware.

## Features

- Distributed inference across multiple machines
- BitTorrent-style collaborative computation
- Fine-tuning and inference up to 10x faster than offloading
- Runs large models on commodity hardware
- Open network of contributors
- Support for popular LLM architectures
