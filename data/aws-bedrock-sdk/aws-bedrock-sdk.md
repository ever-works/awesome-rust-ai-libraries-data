## Overview

The AWS Bedrock SDK is the official Rust integration for Amazon Bedrock, enabling invocation of foundation models from multiple providers including Anthropic Claude and AI21 Labs through a unified AWS service interface.

## Features

- Access to multiple foundation models via AWS Bedrock
- Supports Anthropic Claude, AI21, and other providers
- Official AWS SDK integration
- Enterprise-grade security and IAM integration
- Managed model hosting via AWS
