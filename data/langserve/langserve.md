## Overview

LangServe simplifies the deployment of LangChain applications by converting them into production-ready REST APIs.

## Features

- Automatic REST API generation from LangChain chains
- Schema inference
- Efficient API endpoints
- Playground page for testing
- One-click deployment plans (hosted version)

## Use Cases

- Deploying LangChain applications
- Creating AI-powered APIs
- Rapid prototyping of LLM services
