## Overview

NumKong provides fast, SIMD-optimized computations for common vector distance metrics and similarity functions.

## Features

- L1, L2, Lp distances
- Cosine similarity
- Jaccard similarity
- Hamming distance
- Support for AVX2, AVX-512, NEON

## Use Cases

- Approximate nearest neighbors
- Recommendation systems
- Clustering algorithms

Free and open-source.
