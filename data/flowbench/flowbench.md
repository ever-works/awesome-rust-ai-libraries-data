## Overview

FlowBench is a benchmark presented at EMNLP 2024 for revisiting and evaluating workflow-guided planning capabilities of LLM-based agents. It assesses how well agents can follow, plan, and execute complex multi-step workflows.

## Features

- Workflow-guided planning evaluation for LLM agents
- Presented at EMNLP 2024
- Tests multi-step task planning and execution
- Standardized workflow evaluation protocols
- Assesses agent ability to follow complex procedures

## Use Cases

- Evaluating LLM agent workflow planning
- Benchmarking multi-step reasoning in agents
- Assessing procedural task execution capabilities
