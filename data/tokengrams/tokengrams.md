## Overview

Tokengrams is a tool developed by EleutherAI for computing and storing token n-grams from large text corpora. It aids in dataset analysis by revealing token-level patterns, co-occurrence statistics, and frequency distributions in training data.

## Features

- Computes token n-grams from large text corpora
- Efficient storage and retrieval of n-gram statistics
- Designed for dataset analysis in language model research
- Supports analysis of token-level patterns and frequencies
