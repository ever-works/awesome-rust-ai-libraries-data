## Overview

Haystack is an end-to-end NLP framework by deepset that enables developers to build applications powered by semantic search, question answering, and language agents.

## Features

- Document processing for various file formats and data types
- Neural search powered by transformer models
- Question answering for direct responses to user queries
- Semantic search for meaning-based retrieval
- Agent capabilities for complex multi-step workflows
