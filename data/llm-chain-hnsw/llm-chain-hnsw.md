## Overview

llm-chain-hnsw integrates the llm-chain library with HNSW (Hierarchical Navigable Small World) vector search. It enables semantic retrieval of context for prompts, a key component of Retrieval-Augmented Generation (RAG) workflows.

## Features

- Integration of llm-chain with HNSW vector search
- Semantic context retrieval for LLM prompts
- Approximate nearest neighbor search for fast document retrieval
- Supports RAG pipelines
