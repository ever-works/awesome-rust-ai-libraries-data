## Overview

CAMEL (Communicative Agents for Mind Exploration) is a role-playing framework that enables multi-agent conversations for solving and studying complex tasks.

## Features

- Role-playing framework for agent interaction
- Task-oriented dialogue for goal-directed conversations
- Multi-agent conversations for collaborative problem solving
- Behavioral analysis for studying agent interactions
- Cognitive architecture for understanding agent reasoning
