## Overview

rust-stop-words provides curated lists of common stopwords in various languages. These lists are used to filter out high-frequency words that typically carry little semantic meaning, a standard preprocessing step in NLP pipelines.

## Features

- Stopword lists for multiple languages
- Easy integration into Rust NLP text-processing pipelines
- Useful for text cleaning and feature extraction
- Low-overhead, dependency-free implementation
