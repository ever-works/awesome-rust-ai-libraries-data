## Overview

llm_client is a high-level Rust wrapper around llama.cpp, providing a simplified API for running GPT-style language models locally. It handles the complexity of model loading and inference behind an ergonomic interface.

## Features

- High-level wrapper over llama.cpp with a simple, idiomatic Rust API
- Streaming support for real-time token generation
- Local inference with no external API dependencies
- Supports GPT-style model architectures via llama.cpp backend
