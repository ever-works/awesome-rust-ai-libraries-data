## Overview

OpenGPTs is an open-source project providing customizable GPT-based experiences with full control over AI system components.

## Features

- Three cognitive architectures: Assistant, RAG, and Chatbot
- Control over language models, prompts, tools
- Vector database and retrieval algorithm customization
- Chat history database management
- Deployment via Docker, Cloud Run, and Kubernetes
- Support for various language models

## Pricing

Free and open-source.
