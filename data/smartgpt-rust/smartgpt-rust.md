## Overview

SmartGPT is a Rust framework for building modular LLM-based agents. It separates concerns by providing distinct components for LLMs, tools, memory, and workflow orchestration, enabling the construction of AutoGPT-like autonomous systems.

## Features

- Modular architecture separating LLMs, tools, and memory
- AutoGPT-like system building
- Composable agent components
- Extensible tool integration
