## Overview

Overcooked-AI LLM is a research project that applies LLM multi-agent planning on the OvercookedAI benchmark. It explores how language models can coordinate and cooperate in complex, time-sensitive task environments.

## Features

- LLM multi-agent planning on Overcooked benchmark
- Cooperative task coordination evaluation
- Time-sensitive environment testing
- Language model collaboration assessment

## Use Cases

- Research on LLM cooperative planning
- Evaluating agent coordination capabilities
- Benchmarking multi-agent collaboration
