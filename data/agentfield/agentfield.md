## Overview

AgentField is an open-source infrastructure for AI backends that provides cryptographic identity, guided autonomy, and cross-agent communication capabilities.

## Features

- W3C DIDs for cryptographic agent identity
- Guided autonomy allowing agents to reason freely within policy boundaries
- Async-native execution for workflows running minutes to days
- Cross-agent discovery and RPC without pre-federation
- Built-in memory fabric with vector search
