## Overview

This crate provides Rust bindings to Facebook's fastText library, a widely used tool for learning word embeddings and performing text classification. It bridges Rust applications with the efficient C++ fastText algorithm.

## Features

- Bindings to Facebook's fastText C++ library
- Word embedding generation and usage
- Text classification support
- Efficient training and inference for large corpora

## Pricing

Free and open-source.
