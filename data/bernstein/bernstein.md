## Overview

Bernstein is a deterministic multi-agent orchestrator focused on coordinating AI coding agents in parallel.

## Features

- Spawns parallel AI coding agents (Claude Code, Codex CLI, Gemini CLI)
- Single goal-based orchestration
- Test-based verification
- Auto-commit capabilities
- Zero LLM tokens used for coordination
- Deterministic execution
- Support for multiple coding agent backends

## Pricing

Free and open-source.
