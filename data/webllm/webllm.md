## Overview

WebLLM is a high-performance in-browser LLM inference engine that leverages WebGPU to run large language models directly in the user's browser, bringing LLM capabilities to the edge.

## Features

- In-browser LLM inference using WebGPU
- No server-side computation required
- Support for multiple popular LLM architectures
- Native integration with web applications
- Privacy-preserving (data stays in the browser)
- Powered by TVM Unity compiler
