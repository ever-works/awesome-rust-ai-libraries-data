## Overview

llms-from-scratch-rs is a comprehensive Rust translation of the code examples from Sebastian Raschka's book "Build a Large Language Model from Scratch." It provides a complete, step-by-step implementation of building a large language model from the ground up, translated into Rust.

## Features

- Complete translation of Raschka's book code into idiomatic Rust
- Covers fundamental LLM concepts: tokenization, attention mechanisms, and training
- Educational resource for understanding transformer architecture internals
- Step-by-step implementation from data preparation to model training
- Demonstrates building decoder-only transformer models

## Use Cases

- Learning LLM architecture and training fundamentals through hands-on implementation
- Understanding transformer internals by building them from scratch
- Educational reference for machine learning practitioners moving from Python to Rust
- Complement to Sebastian Raschka's book for Rust developers
