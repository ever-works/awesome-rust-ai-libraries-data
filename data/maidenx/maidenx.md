## Overview

maidenx is a lightweight and fast AI framework in Rust that emphasizes simplicity and performance. It is designed with an educational focus, mirroring PyTorch's architecture to make it approachable for developers familiar with PyTorch.

## Features

- Lightweight design for fast execution
- Educational focus for learning neural network concepts
- Architecture mirrors PyTorch for familiarity
- CPU and CUDA backend support

## Backend Support

- CPU: Host-side computation
- CUDA: NVIDIA GPU acceleration
