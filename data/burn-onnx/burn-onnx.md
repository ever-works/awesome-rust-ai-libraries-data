## Overview

burn-onnx is a dedicated repository for ONNX-related functionality previously part of the main Burn project. It includes crates like burn-import, onnx-ir, onnx-ir-derive, and burn-onnx.

## Key Components

- **burn-import**: Tools for importing PyTorch, SafeTensors, and ONNX models (now recommends burn-store for PyTorch/SafeTensors).
- **onnx-ir and onnx-ir-derive**: Intermediate representation for ONNX.
- **burn-onnx**: ONNX runtime crate.
