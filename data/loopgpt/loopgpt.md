## Overview

LoopGPT is a modular Auto-GPT framework designed for easy deployment and customization of autonomous AI agents.

## Features

- 'Plug N Play' API
- GPT 3.5 compatibility
- Minimal prompt overhead
- Human-in-the-loop capability
- Full state serialization
- Python, CLI, or Docker installation
- Custom tool addition
- Course correction support
- Agent state saving and loading

## Requirements

- Python 3.8+
- OpenAI API Key
- Optional Google search setup
