## Overview

VADER Sentiment is a Rust port of the popular VADER sentiment analysis tool. It uses a rule-based approach combining a valence dictionary of words with their associated sentiments to score text, specifically tuned for social media language including slang, emoticons, and abbreviations.

## Features

- Rule-based sentiment scoring for text
- Tuned for social media language (slang, emoticons, hashtags)
- Produces compound, positive, negative, and neutral sentiment scores
- No ML model dependencies
- Fast and lightweight implementation
