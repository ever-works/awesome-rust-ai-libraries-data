## Overview

8OWLS / WeEvolve is a self-evolving AI agent framework featuring 8 specialized agents organized around the SEED protocol, an 8-phase recursive self-improvement cycle.

## Features

- 8 specialized AI agents with distinct roles
- SEED protocol: 8-phase recursive self-improvement
- Knowledge transfer between agents
- MMORPG-style progression system for agent development
- Supports 15 different models across 7 providers
- Open-source implementation via GitHub

## Models & Providers

Supports 15 models across 7 different provider platforms, enabling flexible model selection based on task requirements.
