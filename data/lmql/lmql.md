## Overview

LMQL is a domain-specific programming language designed specifically for working with large language models, providing native integration of LLMs into code.

## Features

- Python-based syntax
- Conditional logic for LLM operations
- Output constraint specification
- Multi-model support
- Seamless code integration
- Type-safe LLM programming

## Use Cases

- Structured LLM prompting
- Constrained text generation
- Multi-model orchestration
- LLM-powered application development
