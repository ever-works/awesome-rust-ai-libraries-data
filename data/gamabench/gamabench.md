## Overview

GamaBench is a benchmark designed to evaluate the gaming ability of Large Language Models in multi-agent environments. It tests competitive and cooperative reasoning capabilities through various game-based scenarios.

## Features

- Multi-agent gaming evaluation for LLMs
- Tests competitive reasoning capabilities
- Evaluates cooperative agent behavior
- Game-based assessment methodology
- Quantitative metrics for agent gaming performance

## Use Cases

- Benchmarking LLM gaming capabilities
- Evaluating competitive reasoning in agents
- Research on multi-agent game strategies
