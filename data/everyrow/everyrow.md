## Overview

everyrow is an AI-powered data operations SDK designed for applying LLM agents to pandas DataFrames at scale using natural language instructions.

## Features

- Screen, rank, dedupe, and merge rows with natural language
- Web research agents applied per row for enriched data
- Scales to tens of thousands of rows for large datasets
- Claude Code integration for development workflows
