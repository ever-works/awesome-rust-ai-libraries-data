## Overview

OneCLI is an open-source credential vault designed specifically for AI agents. It acts as a Rust-built HTTP gateway that intercepts agent requests and injects API credentials transparently, eliminating the need for agents to handle raw API keys.

## Features

- Transparent API credential injection
- Agents never handle raw API keys or secrets
- Rust-based HTTP gateway for performance
- Reduces credential leakage and exposure risk
- Lightweight and deployable as a sidecar
- MIT licensed
