## Overview

Mobile-Bench is an evaluation benchmark specifically designed for LLM-based mobile agents. It assesses agents' capability to perform tasks on mobile device interfaces and interact with mobile applications.

## Features

- Evaluation benchmark for LLM mobile agents
- Tests mobile UI understanding and interaction
- Mobile-specific task scenarios
- Assesses app navigation capabilities

## Use Cases

- Benchmarking mobile navigation agents
- Evaluating agents on mobile device tasks
- Research on mobile agent capabilities
