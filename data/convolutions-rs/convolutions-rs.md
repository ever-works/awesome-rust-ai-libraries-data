## Overview

`convolutions-rs` is a pure Rust implementation of convolution operations that aims for API compatibility with PyTorch's convolution layers. This allows developers familiar with PyTorch to more easily translate their models to Rust.

## Features

- Pure Rust implementation with no external dependencies
- Torch/PyTorch-compatible API for convolution operations
- 1D, 2D, and 3D convolution support
- Suitable for porting PyTorch models to Rust
