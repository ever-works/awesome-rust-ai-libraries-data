## Overview

EvoAgentX is a framework designed to build self-evolving AI agent ecosystems that can optimize their own workflows and improve over time.

## Features

- Easy Agent and Workflow Customization
- Workflow Optimization & Self-Evolving capabilities
- Agent/workflow evolution algorithms integrated
- Execution Toolkit for running agents and workflows
