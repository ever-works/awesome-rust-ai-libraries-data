## Overview

MInference is a research-backed inference optimization system that significantly speeds up long-context LLM inference through approximate and dynamic sparse attention computation.

## Features

- Approximate and dynamic sparse attention calculation
- Up to 10x inference latency reduction for pre-filling on A100
- Maintains accuracy while reducing computation
- Supports long-context LLM inference
- NeurIPS'24 Spotlight paper
- ICLR'25 publication
