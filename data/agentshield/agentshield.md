## Overview

AgentShield is an open-source security scanner specifically designed for AI agent skills, Model Context Protocol (MCP) servers, and plugins.

## Features

- 30 built-in security rules
- Abstract Syntax Tree (AST) analysis
- Cross-file security tracking
- 5-dimension scoring system
- Zero installation requirement
- 100% offline operation
- MIT licensed
- Available on npm
