## Overview

The `cntk` crate provides Rust bindings for the Microsoft Cognitive Toolkit (CNTK), a unified deep-learning toolkit designed to train and deploy deep learning models. It allows Rust developers to leverage CNTK's inference and training capabilities.

## Features

- Wrapper around Microsoft CNTK library
- Deep learning model loading and inference
- Access to CNTK's neural network operations
- Integration with Rust's type system
