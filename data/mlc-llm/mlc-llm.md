## Overview

MLC LLM is a machine learning compiler and high-performance deployment engine for large language models. Its mission is to enable everyone to develop, optimize, and deploy AI models natively on their devices.

## Features

- Universal LLM deployment via ML compilation
- Native deployment on mobile (iOS, Android), web (WebGPU), and desktop
- Support for diverse hardware backends (CPU, GPU, NPU)
- TVM-based compilation pipeline
- Pre-compiled model libraries for popular architectures
