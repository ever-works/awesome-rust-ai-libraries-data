## Overview

PharmaSimText-OS-LLMs is a repository that includes benchmarks and agents for pharmaceutical simulation testing with LLMs. It evaluates agents' capabilities in healthcare and pharmaceutical domain task execution.

## Features

- Pharmaceutical simulation benchmark
- Healthcare LLM agent evaluation
- Domain-specific task assessment
- Benchmark and agent framework included

## Use Cases

- Benchmarking healthcare AI agents
- Evaluating pharmaceutical domain capabilities
- Research on healthcare task automation
