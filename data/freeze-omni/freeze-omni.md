## Overview

Freeze-Omni is a speech-to-speech dialogue model that achieves low-latency voice interactions using a frozen LLM. It enables real-time conversational audio input and output without unfreezing model weights.

## Features

- Speech-to-speech dialogue capability
- Low-latency audio interactions
- Frozen LLM architecture for efficiency
- Real-time conversational AI
- End-to-end audio processing
