## Overview

Diplomacy-LLM is a public benchmark for evaluating LLM capabilities using results from Diplomacy games played by multiple LLM agents. The complex board game Diplomacy requires negotiation, alliance-building, and strategic planning, making it an ideal testbed for multi-agent LLM evaluation.

## Features

- Diplomacy game benchmark for LLMs
- Multi-agent negotiation evaluation
- Strategic planning assessment
- Alliance-building capability testing
- Public benchmark results and methodology
