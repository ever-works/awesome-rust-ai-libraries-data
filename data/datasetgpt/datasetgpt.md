## Overview

DatasetGPT is a CLI tool for generating textual and conversational datasets using Large Language Models. It automates the creation of training and evaluation data for natural language processing tasks.

## Features

- Command-line interface for dataset generation
- Textual and conversational data creation
- Prompt-based dataset customization
- Automated data generation workflows
- Supports multiple output formats

## Use Cases

- Generating conversational training data
- Creating text datasets for NLP tasks
- Research on LLM-generated datasets
