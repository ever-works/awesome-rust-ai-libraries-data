## Overview

rust-genai is a multi-provider generative AI SDK for Rust. It supports a wide range of providers including OpenAI, Anthropic, xAI, Groq, Ollama, and more, all accessible through a single unified interface.

## Features

- Support for OpenAI, Anthropic, xAI, Groq, Ollama, and more
- Single interface for many AI services
- Provider-agnostic application design
- Supports chat completions and other LLM endpoints

Experimental.
