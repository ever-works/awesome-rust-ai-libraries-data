## Overview

LibreEval is an open-source benchmark designed specifically for detecting hallucinations in Retrieval-Augmented Generation (RAG) systems. It provides standardized evaluation methodologies to assess the accuracy and reliability of RAG pipelines.

## Features

- Hallucination detection for RAG systems
- Open-source benchmarking methodology
- Standardized evaluation protocols
- Assesses RAG pipeline accuracy
- Detects generated content reliability issues

## Use Cases

- Evaluating RAG system accuracy
- Benchmarking hallucination rates
- Research on RAG system reliability
